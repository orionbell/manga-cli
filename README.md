# manga-cli

A bash script that lets you read manga without leaving the terminal!


## Dependencies
- [fzf](https://github.com/junegunn/fzf)
- [jq](https://github.com/jqlang/jq)
- [kitty](https://github.com/kovidgoyal/kitty)
